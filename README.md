# Community Issue Reporting System

## Overview
This application is a **Windows Forms C# project** that allows users to report issues within their local community.  
It forms Part 1 of the POE and implements the **Report Issues** feature.

## Features
- **Main Menu (Form)** – Startup menu that provides access to different sections (Report Issues, etc).
- **Report Issues Form** – Users can submit details about problems in their area.
- **File Attachments** – Option to attach images or documents for better issue context.
- **Feedback Messages** – Informative messages that guide the user on successful or invalid submissions.
- **Engagement Feature** – A progress bar and encouragement text help users provide complete, high-quality reports.

## Build and Run

1. Launch the application in Visual Studio (C# .NET Windows Forms).
2. Set the startup project to MunicipalServicesApp.
3. Run the application (F5 or Ctrl+F5).
4. The **Main Menu** will appear – select "Report Issues."
5. Fill in the following fields:
   - Location
   - Category
   - Description (must be detailed, >25 characters recommended)
   - Attachment (optional, but encouraged)
6. Watch the **progress bar** update as you provide more details.
7. Submit the issue – feedback messages confirm success or errors.

## Requirements
- Visual Studio 2019/2022
- .NET Framework (Windows Forms project type)
